# YouTube search fixtures

The synthetic fixtures in this directory are deliberately minimized structures
derived from the search traversal behavior in pinned yt-dlp reference commit
`aefce1eea4d0b6bab1ec2bd3beff09bff91a39c8`, specifically
`yt_dlp/extractor/youtube/_search.py` and `_tab.py::_search_results`.
They contain invented IDs, titles, continuation tokens, and client values; no
live response or credential is recorded.
